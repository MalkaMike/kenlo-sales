# Kenlo Design System - Key Findings

**Source:** Kenlo-DesignBranding.pdf (Setembro 2021)

## 1. Logo & Área de Proteção
- Área livre = largura da letra "n" de Kenlo (1x acima, abaixo, esquerda, direita)
- Logo institucional: usar cores principais sempre que possível
- Variações: vermelho (#F82E52), preto (#242424), branco

## 2. Cores Principais
- **Vermelho (Primary):** #F82E52
- **Preto:** #242424

## 3. Cores de Apoio
- **Cinza:** #D8D8D8
- **Cinza claro:** #F2F2F2
- **Verde:** #4ABD8D

## 4. Tipografia
- **Fonte:** Manrope (Google Fonts, designed by Mikhail Sharanda 2018)
- **Uso:** Texto corrido e subtítulos
- **Pesos:** Regular, Bold

## 5. Estilo Fotográfico
- Imagens focadas em **pessoas usando produtos tecnológicos** e suas ações
- Evitar pessoas olhando intencionalmente para câmera (manter espontaneidade)
- **Aumentar temperatura** da foto aplicando filtro de calor nos tons médios e levemente nos brancos
- **Inserir cores da marca** de forma natural, preservando luz e temperatura ajustadas
- Inserção deve acontecer sobre os elementos mais coloridos da fotografia

## 6. Princípios de Design
- Versão com cores institucionais é a principal forma de apresentação
- Usar sempre que possível
- Manter consistência visual em todos os materiais

## 7. Aplicações para Portal de Vendas
- **Backgrounds:** Usar cinza claro (#F2F2F2) ou branco para fundos
- **Destaques:** Vermelho (#F82E52) para números-chave, CTAs, badges
- **Boxes/Cards:** Bordas sutis, shadows leves, usar cinza (#D8D8D8) para separadores
- **Tipografia:** Manrope Regular para corpo, Bold para títulos e números importantes
- **Verde (#4ABD8D):** Usar para indicadores positivos, crescimento, "incluído"
- **Preto (#242424):** Usar para textos principais quando não for vermelho

## 8. Estilo Visual Recomendado
- **Informativo:** Dados claros, hierarquia visual bem definida
- **Premium:** Uso estratégico de espaço branco, shadows sutis, bordas limpas
- **Boxes:** Cards com bg-white, border-gray-200, shadow-sm, rounded-lg
- **Não usar:** Gradientes excessivos, sombras pesadas, efeitos "teatrais"
- **Usar:** Estrutura limpa, dados em destaque, boxes organizados
